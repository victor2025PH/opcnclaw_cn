## Windows 安装 <a name="windows"></a>

### 方式一：一键自动安装（推荐 99% 用户使用）

**双击 `install_full.bat` 即可**

安装程序会自动完成：
1. ✅ 检测/下载 Python 3.11（如未安装自动下载）
2. ✅ 安装 Visual C++ 运行库（torch 必需）
3. ✅ 配置国内 pip 镜像（阿里云，速度快）
4. ✅ 安装所有 Python 依赖
5. ✅ 生成 SSL 证书（麦克风权限必需）
6. ✅ 创建桌面快捷方式
7. ✅ 创建开始菜单项

**预计时间：** 10-30 分钟（取决于网速，torch 约 500MB）

---

### 方式二：Inno Setup 安装包（专业打包，可分发）

> 适合制作可以发给他人安装的 `.exe` 文件

**步骤：**
1. 先运行 `install_full.bat` 完成依赖安装
2. 下载安装 [Inno Setup 6](https://jrsoftware.org/isdl.php)
3. 双击 `installer.iss` → 点击"编译"按钮
4. 得到 `dist\installer\OpenClaw-v2.0-Setup.exe`

> 生成的 `Setup.exe` 可以直接发给任何人，双击即装

---

### 方式三：便携包（无需安装，U盘即用）

```
双击 build.bat → 选择 [1] 快速便携包
```

生成 `dist\OpenClaw-v2.0\` 文件夹，整个文件夹复制到任何 Windows 电脑即可使用。

## macOS 安装 <a name="macos"></a>

> 支持 macOS 12+ ，Intel 和 Apple Silicon (M1/M2/M3) 均可

### 步骤：

**第一步：下载项目文件**

将整个项目文件夹放到你的 Mac 上（如 `/Users/你的用户名/OpenClaw`）

**第二步：打开终端运行安装脚本**

```bash
# 打开终端（Spotlight 搜索 "终端" 或 Terminal）
cd /你的项目路径

# 给脚本添加执行权限
chmod +x 一键安装-Mac.sh

# 运行安装脚本
./一键安装-Mac.sh
```

**安装程序会自动完成：**
1. ✅ 安装 Homebrew（如未安装）
2. ✅ 安装/检测 Python 3.11+
3. ✅ 安装系统依赖（ffmpeg、portaudio）
4. ✅ 创建 Python 虚拟环境
5. ✅ 安装所有依赖（Apple Silicon 自动使用 MPS 加速）
6. ✅ 引导配置 API Key
7. ✅ 预下载 Whisper 语音模型（145MB）
8. ✅ 创建桌面快捷方式 + 开机自启

**预计时间：** 15-40 分钟（首次安装 Homebrew 较慢）

---

### macOS 权限说明

首次使用时 macOS 会弹窗请求以下权限，**全部点"允许"：**

| 权限 | 用途 |
|------|------|
| 🎙️ 麦克风 | 语音识别 |
| 📷 摄像头（可选）| 手势控制 |
| 🖥️ 屏幕录制（可选）| 远程桌面功能 |
| ⌨️ 辅助功能（可选）| 远程鼠标键盘控制 |

如果弹窗消失了，手动开启：  
**系统设置 → 隐私与安全性 → 麦克风 → 找到 Terminal/终端 → 打开**

### 配置方式（二选一）

**方式A：图形界面（推荐）**
1. 启动 OpenClaw 后，右键点击任务栏图标 → 设置
2. 点击"AI 平台"标签

```
# Mac: open -e ~/.env 或 nano ~/OpenClaw/.env

OPENCLAW_GATEWAY_TOKEN=你的OpenClaw Token
```

## 日常使用 <a name="日常使用"></a>

### Windows 启动
- 双击桌面 **「OpenClaw AI助手」** 图标
- 或运行 `start.bat`

### macOS 启动
- 双击桌面 **「OpenClaw AI.command」** 图标
- 服务开机自动启动，无需每次手动开

### 访问界面

| 地址 | 用途 |
|------|------|
| `http://localhost:8766/app` | 电脑完整版（推荐） |
| `http://localhost:8766/qr` | 扫码连接手机 |
| `https://本机IP:8765/app` | HTTPS 版（麦克风需要） |

### 手机使用
1. 确保手机和电脑在同一 Wi-Fi
2. 打开 `http://本机IP:8766/qr`（电脑上）
3. 用手机扫码
4. 手机浏览器中点击"添加到主屏幕"（PWA）

## 常见问题 <a name="常见问题"></a>

### ❓ Windows 安装后双击图标没反应
**解决：** 右键 `install_full.bat` → 以管理员身份运行

### ❓ 出现"No module named torch"错误
**解决：**
```bat
# Windows 命令行中运行：
pip install torch --index-url https://download.pytorch.org/whl/cpu
```

### ❓ Mac 提示"无法打开，因为无法验证开发者"
**解决：** 系统设置 → 隐私与安全性 → 仍要打开

### ❓ 麦克风不工作（说话没反应）
**Windows 解决：**
1. Windows 设置 → 隐私 → 麦克风 → 允许应用访问麦克风
2. 重启 OpenClaw

**macOS 解决：**  
系统设置 → 隐私与安全性 → 麦克风 → 打开 Terminal 的权限

### ❓ AI 不回复（转圈）
检查 API Key 是否配置正确，或尝试切换其他 AI 平台。

### ❓ 语音识别中文不准
在设置面板中将 STT 模型从 `base` 改为 `small` 或 `medium`。

### ❓ 端口 8765/8766 被占用
**Windows：**
```bat
netstat -ano | findstr :8766
taskkill /PID <进程ID> /F
```
**macOS：**
```bash
lsof -i :8766
kill -9 <PID>
```

## 卸载 <a name="卸载"></a>

### Windows 卸载
1. 控制面板 → 程序 → 卸载 → OpenClaw AI 助手
2. 或直接删除安装目录文件夹

### macOS 卸载
```bash
# 停止服务
launchctl unload ~/Library/LaunchAgents/com.openclaw.voice.plist

# 删除文件
rm -rf ~/OpenClaw
rm ~/Library/LaunchAgents/com.openclaw.voice.plist
rm ~/Desktop/"OpenClaw AI.command"
```
