## 三、安装方法

### Windows 安装

#### 方式一：一键自动安装（推荐）

**双击 `install_full.bat` 即可**，安装程序自动完成：

1. ✅ 检测/下载 Python 3.11（如未安装自动下载）
2. ✅ 安装 Visual C++ 运行库（torch 必需）
3. ✅ 配置国内 pip 镜像（阿里云）
4. ✅ 安装所有 Python 依赖
5. ✅ 生成 SSL 证书（手机麦克风权限必需）
6. ✅ 创建桌面快捷方式和开始菜单项

**预计时间：** 10–30 分钟（torch 约 500MB）

#### 方式二：Inno Setup 安装包（可分发）

适合制作可以发给他人的 `.exe` 安装文件：

1. 先运行 `install_full.bat` 完成依赖安装
2. 下载安装 [Inno Setup 6](https://jrsoftware.org/isdl.php)
3. 双击 `installer.iss` → 点击"编译"
4. 得到 `dist\installer\OpenClaw-v3.2-Setup.exe`

#### 方式三：便携包（U 盘即用）

```
双击 build.bat → 选择 [1] 快速便携包
```

生成的 `dist\OpenClaw-v3.2\` 文件夹可复制到任何 Windows 电脑直接使用。

---

### macOS 安装

```bash
cd /你的项目路径
chmod +x 一键安装-Mac.sh
./一键安装-Mac.sh
```

安装程序自动完成：

1. ✅ 安装 Homebrew（如未安装）
2. ✅ 安装 Python 3.11+、ffmpeg、portaudio
3. ✅ 创建虚拟环境并安装依赖（Apple Silicon 自动使用 MPS 加速）
4. ✅ 引导配置 API Key
5. ✅ 预下载 Whisper 语音模型（145MB）
6. ✅ 创建桌面快捷方式 + 开机自启

**macOS 权限：** 首次使用时会弹窗请求麦克风、摄像头、屏幕录制、辅助功能权限，全部允许即可。

### AI 自动轮换

配置多个平台后，系统自动处理：
- 某平台限速（429 错误）→ 自动切换下一个
- 余额耗尽 → 自动降级到免费平台
- 网络错误 → 60 秒冷却后重试

调度模式（`config.ini` 中 `[router] mode`）：

| 模式 | 说明 |
|------|------|
| `cost_saving` | 省钱模式（默认），优先用免费平台 |
| `quality_first` | 质量优先，选最好的模型 |
| `speed_first` | 速度优先，多平台赛跑取最快 |


### 配置文件

#### config.ini（主配置）

```ini
[router]
mode = cost_saving          # 调度模式: cost_saving / quality_first / speed_first
auto_switch = true          # 平台自动切换
cooldown_seconds = 60       # 错误冷却时间
max_errors = 3              # 最大连续错误数

[providers]
order = openclaw,zhipu_flash,baidu_speed,siliconflow_free,deepseek,tongyi

[stt]
model = base                # 语音识别模型: tiny / base / small / medium
language = zh               # 识别语言: zh / en / auto
device = cpu                # 计算设备: cpu / cuda / mps / auto

[tts]
provider = edge_tts         # TTS 引擎: edge_tts / elevenlabs / chatterbox
voice = zh-CN-XiaoxiaoNeural

[system]
http_port = 8766            # HTTP 端口
https_port = 8765           # HTTPS 端口
autostart = false           # 开机自启
minimize_to_tray = true     # 最小化到托盘

[ui]
theme = dark                # 主题: dark / light
language = zh               # 界面语言: zh / en
```

#### .env（密钥和高级配置）

```ini
# AI 网关
OPENCLAW_GATEWAY_URL=http://127.0.0.1:18789
OPENCLAW_GATEWAY_TOKEN=

# OpenAI 兼容
OPENAI_API_KEY=
OPENAI_BASE_URL=
OPENAI_MODEL=

# 智谱视觉
ZHIPU_VISION_API_KEY=
ZHIPU_VISION_MODEL=glm-4v-flash

# 语音
OPENCLAW_STT_MODEL=base
OPENCLAW_STT_DEVICE=cpu

# TTS
EDGE_TTS_VOICE=zh-CN-XiaoxiaoNeural

# 安全
OPENCLAW_REQUIRE_AUTH=false

# 日志
OPENCLAW_LOG_LEVEL=DEBUG
OPENCLAW_LOG_ROTATION=5 MB
OPENCLAW_LOG_RETENTION=14 days
```

### 启动流程（技术细节）

1. 配置 UTF-8 编码（Windows 兼容）
2. 初始化 loguru 日志系统（控制台 + 文件轮转）
3. 加载 `config.ini` 路由配置
4. 初始化 AI 路由器（多平台自动切换）
5. 加载 63 个技能引擎
6. 启动系统托盘图标（`--nogui` 模式跳过）
7. 注册全局热键：`Ctrl+Shift+O`（设置）、`Ctrl+Shift+X`（退出）
8. 启动上下文感知引擎
9. 启动 FastAPI 双端口服务器（HTTP:8766 + HTTPS:8765）
10. 打开浏览器或显示首次向导

| STT 模型 | 速度 | 准确率 | 推荐场景 |
|----------|------|-------|---------|
| tiny | <1 秒 | 70% | 不推荐中文 |
| base | 1–2 秒 | 85% | **默认，日常使用** |
| small | 2–3 秒 | 90% | 准确率要求高 |
| medium | 3–5 秒 | 95% | **最佳效果** |

- `zh-CN-XiaoxiaoNeural` — 晓晓（女，活泼，默认）

### 安装相关

**❓ Windows 安装后双击图标没反应**
→ 右键 `install_full.bat` → 以管理员身份运行

**❓ 出现 "No module named torch"**
→ 命令行运行：`pip install torch --index-url https://download.pytorch.org/whl/cpu`

**❓ 安装依赖下载失败**
→ 关闭 VPN/代理，手动运行：`pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/`

**❓ Mac 提示"无法打开，因为无法验证开发者"**
→ 系统设置 → 隐私与安全性 → 仍要打开

### 使用相关

**❓ 麦克风不工作**
- 检查浏览器地址栏左侧锁形图标 → 允许麦克风
- HTTP 下 Chrome/Safari 不允许调麦克风 → 改用 `https://本机IP:8765/app`
- Windows：系统设置 → 隐私 → 麦克风 → 允许应用访问
- macOS：系统设置 → 隐私与安全性 → 麦克风 → 允许 Terminal

**❓ 语音识别不准确**
→ 设置 → 语音 → STT 模型改为 `small` 或 `medium`

**❓ AI 不回复（转圈）**
→ 检查 API Key 是否配置正确，或添加更多平台（自动轮换）

**❓ AI 回复很慢**
→ 设置调度模式为 `speed_first`；推荐添加 Groq 平台（速度极快）

**❓ 端口 8765/8766 被占用**

Windows：
```cmd
netstat -ano | findstr :8766
taskkill /PID <进程ID> /F
```

macOS：
```bash
lsof -i :8766
kill -9 <PID>
```

**❓ 如何查看日志**
- Windows：`logs\openclaw.log`
- macOS：`~/OpenClaw/logs/stdout.log`
- 调试模式：运行 `openclaw_debug.bat`

## 八、卸载方法

### Windows

**方式 1：** 控制面板 → 程序 → 卸载 → OpenClaw AI 助手

**方式 2：** 直接删除安装目录文件夹

### macOS

```bash
launchctl unload ~/Library/LaunchAgents/com.openclaw.voice.plist
rm -rf ~/OpenClaw
rm ~/Library/LaunchAgents/com.openclaw.voice.plist
rm ~/Desktop/"OpenClaw AI.command"
```
