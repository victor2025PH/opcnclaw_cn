OpenClaw AI

iPhone / iPad 设置向导

0 用 iPhone 扫描二维码

打开 iPhone 摄像头，扫描下方二维码开始设置。

1 下载安全证书

iPhone 需要信任此证书才能安全地使用语音、摄像头和手势功能。

下载证书配置文件 下载 .crt 文件 (备选)

2 安装证书

下载后，在 iPhone 上按以下步骤操作：

  • a 会出现通知："已下载描述文件"。点击它，或打开 设置 App。
  • b 前往 设置 → 通用 → VPN与设备管理（或 描述文件与设备管理）
  • c 点击 "OpenClaw AI - Secure Connection" → 点击 安装 → 输入密码 → 再次点击 安装。

3 启用完全信任

此步骤让 iPhone 完全信任该证书以建立 HTTPS 连接：

  • a 前往 设置 → 通用 → 关于本机 → 下滑到 证书信任设置
  • b 找到 "OpenClaw Local CA" 并打开开关
  • c 在弹出的警告对话框中点击 继续。

4 启动应用

设置完成！点击下方按钮，开始使用语音、摄像头和手势功能。

证书仅在本地网络有效。
此设置只需进行一次。
