## 一、环境准备

### 1.1 启动服务

```bat
cd D:\xlx2026\openclaw-voice
start.bat
```

或无 GUI 模式：

```bat
python launcher.py --nogui
```

### 1.2 预期启动日志

```
正在启动 OpenClaw AI 语音助手...
Initializing OpenClaw Voice server...
Loading STT model: base
Loading TTS model: chatterbox
Uvicorn running on https://0.0.0.0:8765
Uvicorn running on http://0.0.0.0:8766
```

### 1.3 测试页面

| 页面 | 地址 | 说明 |
|------|------|------|
| 主入口 | http://localhost:8766/ | index.html |
| **聊天页（重点）** | **http://localhost:8766/app/** | **app.html — 所有 R11-R16 功能在此页** |
| 管理页 | http://localhost:8766/admin/ | admin.html |

### 1.4 测试工具

- 浏览器：Chrome / Edge（推荐）
- 按 **F12** 打开 DevTools → **Console** 标签页，全程监控 JS 错误
- 准备 1-2 个测试文件（图片 + 非图片）用于拖拽上传测试

### 1.5 后端依赖说明

| 依赖 | 影响的功能 | 缺失时表现 |
|------|-----------|-----------|
| AI API（OpenAI/GLM） | 内联翻译、对话摘要、消息编辑重发 | 调用失败，toast 报错 |
| Ollama | 模型基准测试 | 测试按钮调用失败 |
| TTS 服务 | TTS 试听 | 播放失败 |
| 麦克风权限 | 语音录制、音频可视化、实时字幕 | 无法录音 |

> **提示**: 即使后端未完全配置，大部分 UI 交互仍可验证（面板是否显示、按钮是否响应、样式是否正确）。
