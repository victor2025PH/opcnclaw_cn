# OpenClaw AI 语音助手 — 简版使用手册

> 一句话介绍：OpenClaw 是一个装在你电脑上的 AI 语音助手，你可以跟它说话、打字聊天，它还能帮你查天气、算账、写东西、控制电脑。

---

## 怎么装？

### Windows 电脑

**运行安装包（OpenClaw-v3.3-Setup.exe），一路点「下一步」到「完成」即可。**

安装程序会自动完成：下载并配置 Python、安装所有依赖包、生成证书。全程约 5–15 分钟（需联网），装完后桌面会出现「OpenClaw 增强版」图标，双击即可使用，无需再运行其他任何东西。

> 若没有安装包：在已解压的目录里双击 `install_full.bat` 也可完成安装。若双击没反应，可右键该文件 → 选「以管理员身份运行」。

### 苹果电脑 (Mac)

打开"终端"程序（在启动台里搜"终端"），输入：

```
cd /你放文件的路径
./一键安装-Mac.sh
```

按提示操作即可，装完桌面会出现快捷方式。

## 怎么启动？

**Windows：** 双击桌面上的「OpenClaw AI 助手」图标

**Mac：** 双击桌面上的「OpenClaw AI.command」图标

启动后几秒钟，电脑右下角会出现一个小图标，浏览器会自动打开聊天界面。

如果浏览器没自动打开，手动打开浏览器访问：**http://localhost:8766/app**

**桌面图标双击没反应？**

- 若您是用安装包安装的，安装程序已自动装好运行环境和依赖，一般可直接使用。若仍无反应：点开始菜单 → 找到「OpenClaw 增强版」下面的 **「命令行模式（调试）」**，点开会弹出黑窗口并显示报错，便于排查。
- 安装目录（如 `C:\Program Files (x86)\OpenClaw`）里的 **openclaw_debug.bat** 双击也可查看报错。

## 第一次用，必须先做这一步

**配置 AI 钥匙（不配就没法聊天）**

你需要去 AI 平台注册一个账号，拿到一个"钥匙"（一串字母数字）。

**推荐用智谱 AI，永久免费：**

1. 打开 https://open.bigmodel.cn ，注册一个账号
2. 登录后点左边的"API Keys"
3. 点"创建 API Key"，复制那串字符
4. 回到 OpenClaw，点右上角 ⚙️ 齿轮图标 → 找到"AI 平台" → 把钥匙粘贴进去 → 保存

**搞定！现在可以正常聊天了。**

> 其他免费平台也行：硅基流动（cloud.siliconflow.cn）、DeepSeek（platform.deepseek.com）

## 聊天界面长什么样？

打开后你会看到一个聊天窗口，和微信聊天差不多：

- **底部输入框** → 打字发消息，按回车发送
- **底部麦克风按钮 🎙️** → 点一下开始说话，说完它会自动识别
- **右上角一排小按钮** → 各种功能入口（下面详细说）

> 小屏幕（手机/平板）上按钮会自动精简，多出来的功能藏在右上角 **⋮** 三个点的菜单里。

## 有哪些功能？

### 💬 聊天对话

最基本的功能，打字或说话跟 AI 聊天。

| 你可以这样做 | 怎么操作 |
|-------------|---------|
| 打字聊天 | 底部输入框打字，按回车发送 |
| 语音聊天 | 点麦克风按钮，说话就行 |
| 发图片让 AI 看 | 点输入框左边的 📎 回形针，选图片 |
| 拖文件进去 | 直接把文件拖到聊天窗口里 |

### ⭐ 消息管理

| 功能 | 怎么用 |
|------|-------|
| 收藏消息 | 鼠标放到消息上，点 ⭐ 星标 |
| 编辑已发的消息 | 鼠标放到你的消息上，点 ✏️ 铅笔 |
| 固定消息到顶部 | 鼠标放到消息上，点 📌 图钉 |
| 翻译消息 | 鼠标放到消息上，点 🌐 地球 |
| 搜索聊天记录 | 按键盘 Ctrl+F |
| 导出聊天记录 | 右上角 📥 导出按钮 |
| 对话摘要 | 右上角 📋 摘要按钮，AI 帮你总结 |

### 🧩 技能中心（63 个实用技能）

点右上角 🧩 四方格图标打开。不用记技能名，直接说你要什么就行：

**日常生活：**
- "现在几点了？"
- "100 美元换多少人民币？"
- "帮我生成一个密码"
- "今天北京天气怎么样？"

**算钱：**
- "贷款 100 万 30 年月供多少？"
- "月薪 2 万税后到手多少？"
- "100 加 50 乘以 3 等于多少？"

**健康：**
- "我身高 170 体重 70，BMI 多少？"
- "每天应该喝多少水？"

**做饭：**
- "冰箱里有鸡蛋和西红柿，能做什么菜？"
- "红烧肉怎么做？"

**写东西：**
- "帮我写一封请假邮件"
- "帮我想个朋友圈文案"
- "帮我给小孩取个名字"

**还有更多：** 笑话、猜谜、翻译、诗词、星座、交规查询……

### 📷 摄像头和手势控制

点右上角 📷 摄像头按钮开启：

- ✋ 张开手掌 → 开始说话
- ✊ 握拳 → 停止
- 👍 大拇指向上 → 向上翻
- 👎 大拇指向下 → 向下翻
- ✌️ 剪刀手 → 拍照

### 🖥️ 远程桌面

点右上角 🖥️ 显示器按钮，可以在手机上看到并操控电脑屏幕。用手指点就是鼠标点击，拖动就是移动鼠标。

### 💬 微信自动回复

点右上角 💬 微信图标，可以让 AI 自动帮你回复微信消息：

1. 打开总开关
2. 添加你要自动回复的联系人
3. 设好之后，有消息来 AI 会自动帮你回

> 可以设置"人工审核"，AI 写好回复先给你看，你确认才发送。

### 🎨 个性化设置

点右上角 ⚙️ 齿轮进入设置：

- **换主题：** 深色/浅色/跟随系统
- **换颜色：** 7 种强调色可选
- **调字体大小：** 拖动滑块
- **换语言：** 中文/英文
- **换 AI 声音：** 多种男女声可选，可以试听

## 手机上怎么用？

1. 手机和电脑连**同一个 WiFi**
2. 电脑浏览器打开 **http://localhost:8766/qr**
3. 用手机扫屏幕上的二维码
4. 手机浏览器打开后，点分享 → **"添加到主屏幕"**

这样手机桌面就会多一个 OpenClaw 图标，打开就能用，跟装了个 App 一样。

> 如果手机上说话功能不好使，把地址换成 **https://电脑IP:8765/app**（第一次会提示不安全，点"继续访问"就行）

## 常见问题

**装完双击图标没反应？**
→ 右键 `install_full.bat`，选"以管理员身份运行"，重新装一遍

**说话没反应？**
→ 浏览器地址栏左边有个锁或ℹ️图标，点它 → 把麦克风改成"允许" → 刷新页面

**AI 不说话了？**
→ 检查有没有配 AI 钥匙（第一次用必须配，看上面的步骤）

**中文识别不准？**
→ 点设置 ⚙️ → 语音 → 把 STT 模型从 base 改成 small 或 medium（更准但稍微慢一点）

**怎么关掉？**
→ 电脑右下角找到 OpenClaw 小图标，右键 → 退出

## 怎么卸载？

**Windows：** 控制面板 → 程序 → 卸载 → 找到 OpenClaw → 卸载。或者直接删掉安装文件夹。

**Mac：** 删掉 `~/OpenClaw` 文件夹和桌面上的快捷方式就行。
