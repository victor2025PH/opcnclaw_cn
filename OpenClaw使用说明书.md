## 1. 软件简介

OpenClaw 是一款**本地部署**的全双工 AI 语音助手，所有计算在您自己的电脑上完成，对话数据不经过第三方服务器。

### 核心特点

| 特性 | 说明 |
|------|------|
| 🎙️ 全双工对话 | 说话时 AI 可以打断您，您也可以随时打断 AI |
| 🤖 13个AI平台 | 智谱、DeepSeek、硅基流动等，自动轮换不中断 |
| 🧩 63个技能 | 天气、计算、菜谱、财务、健康等实用功能 |
| 📱 手机扫码 | 手机扫码即用，支持添加到主屏幕 |
| 🔒 隐私安全 | 本地运行，语音数据不上传 |
| 🖥️ 远程桌面 | 用手机/语音控制电脑 |
| 🏢 企业集成 | 连接飞书/企微/钉钉群机器人 |

## 3. 安装方法

### Windows 安装

**安装包位置：**
```
D:\xlx2026\openclaw-voice\dist\installer\OpenClaw-v2.0-Setup.exe
```

**安装步骤：**

1. 双击 `OpenClaw-v2.0-Setup.exe`
2. 选择安装语言：**简体中文**
3. 阅读欢迎界面，点击"下一步"
4. 选择安装目录（默认 `C:\Program Files\OpenClaw`）
5. 勾选"创建桌面快捷方式"（推荐）
6. 点击"安装"
7. **等待约 10-30 分钟**（自动下载安装 Python 依赖）
8. 安装完成，选择"立即启动"

> ⚠️ 安装过程中会自动安装 Python 3.11 和约 500MB 的 AI 依赖，请保持网络畅通

### macOS 安装

打开终端，执行：
```bash
cd /OpenClaw项目目录
chmod +x 一键安装-Mac.sh
./一键安装-Mac.sh
```

## 6. 语音功能使用

### 开启语音对话

1. 打开网页界面 `http://localhost:8766/app`
2. 允许浏览器麦克风权限（地址栏左侧图标）
3. 点击界面下方**麦克风按钮** 或 说"**你好小龙**"唤醒

### 语音按钮说明

| 操作 | 效果 |
|------|------|
| 单击麦克风 | 开始录音，松开发送 |
| 长按麦克风 | 持续录音 |
| 说"你好小龙" | 唤醒词自动开始录音 |
| 说话时 AI 在说 | 直接打断 AI |

### 语音识别（STT）设置

打开设置 → 语音设置：

| 模型 | 速度 | 准确率 | 推荐场景 |
|------|------|-------|---------|
| tiny | 最快（<1秒）| 70% | 方言/嘈杂环境不推荐 |
| base | 快（1-2秒）| 85% | **默认，日常使用** |
| small | 中（2-3秒）| 90% | 准确率要求高 |
| medium | 慢（3-5秒）| 95% | **最佳，CPU性能够用** |

> 💡 切换模型后需要等待几秒下载/加载，无需重启

### 语音合成（TTS）设置

默认使用 **微软 Edge TTS（免费）**，超过 400 种声音：

常用中文声音：
- `zh-CN-XiaoxiaoNeural` - 晓晓（女，活泼，**默认**）
- `zh-CN-YunxiNeural` - 云希（男，成熟）
- `zh-CN-YunyangNeural` - 云扬（男，新闻播报风格）
- `zh-CN-XiaohanNeural` - 晓涵（女，温柔）
- `zh-CN-XiaomengNeural` - 晓梦（女，甜美）

### 连续对话模式

设置 → 语音 → 开启"连续模式"：
- AI 回复完毕后自动开始监听
- 适合长时间语音交互，不需要反复点击

## 8. 手机使用（扫码/PWA）

### 手机访问步骤

1. 电脑和手机连接**同一个 Wi-Fi**
2. 在电脑浏览器打开 `http://localhost:8766/qr`
3. 用手机扫描二维码
4. 手机浏览器打开聊天界面

### 添加到手机主屏幕（PWA）

像 App 一样使用，不需要下载安装：

**iPhone / iPad（Safari）：**
1. 用 Safari 打开网页
2. 点击底部分享按钮（方框+箭头）
3. 向下滑动找到"添加到主屏幕"
4. 点击"添加"

**Android（Chrome）：**
1. 用 Chrome 打开网页
2. 点击右上角三点菜单
3. 选择"添加到主屏幕"
4. 点击"添加"

> 添加后桌面会出现 OpenClaw 图标，全屏运行，体验接近原生 App

### 手机版注意事项

- 手机版使用 HTTP（8766端口），麦克风权限需要 HTTPS
- **如需语音功能**，用 `https://电脑IP:8765/app` 访问（HTTPS版）
- 首次访问 HTTPS 版会提示证书不安全，选择"继续访问"即可（自签名证书）

## 9. 企业IM集成（飞书/企微/钉钉）

可以将 AI 对话结果同步推送到企业群聊，方便团队共享。

### 飞书机器人配置

1. 在飞书群聊中添加自定义机器人
2. 飞书群 → 设置 → 机器人 → 添加机器人 → 自定义机器人
3. 复制 Webhook URL
4. OpenClaw 设置 → 平台接入 → 飞书 → 粘贴 URL → 测试连接

### 企业微信机器人配置

1. 企微群 → 群机器人 → 添加机器人 → 自定义
2. 复制 Webhook URL
3. OpenClaw 设置 → 平台接入 → 企业微信 → 粘贴 URL → 测试连接

### 钉钉机器人配置

1. 钉钉群 → 群设置 → 智能群助手 → 添加机器人 → 自定义
2. 复制 Webhook URL，记录安全设置中的"加签密钥"
3. OpenClaw 设置 → 平台接入 → 钉钉 → 粘贴 URL 和密钥 → 测试连接

### 推送效果

配置后，每次 AI 完整回复都会自动推送到对应群聊，格式如下：
```
[OpenClaw AI]
用户：今天北京天气怎么样？
AI：今天北京晴天，气温 8-18℃，西北风3级...
```

## 13. 高级配置（.env 文件）

高级用户可以直接编辑 `.env` 文件进行精细配置。

**文件位置：**
```
C:\Program Files\OpenClaw\.env
```
（或安装时选择的其他目录）

**用记事本打开：** 右键 `.env` → 用记事本打开

### 主要配置项说明

```ini
# ── AI 网关（主推荐） ──────────────────────────────
OPENCLAW_GATEWAY_URL=http://127.0.0.1:18789
OPENCLAW_GATEWAY_TOKEN=你的Token

# ── OpenAI 兼容接口 ────────────────────────────────
OPENAI_API_KEY=sk-xxxxxxxx
OPENAI_BASE_URL=https://api.deepseek.com/v1    # DeepSeek
OPENAI_MODEL=deepseek-chat

# ── 智谱视觉（图片分析） ──────────────────────────
ZHIPU_VISION_API_KEY=你的智谱Key
ZHIPU_VISION_MODEL=glm-4v-flash

# ── 语音识别（STT）设置 ────────────────────────────
OPENCLAW_STT_MODEL=base          # tiny/base/small/medium
OPENCLAW_STT_DEVICE=cpu          # cpu（无GPU时）
OPENCLAW_STT_LANGUAGE=zh         # zh中文 / en英文 / auto自动

# ── 语音合成（TTS）设置 ────────────────────────────
EDGE_TTS_VOICE=zh-CN-XiaoxiaoNeural

# ── 服务器端口 ─────────────────────────────────────
OPENCLAW_PORT=8766               # HTTP端口
                                  # HTTPS默认 8765

# ── 安全设置 ───────────────────────────────────────
OPENCLAW_REQUIRE_AUTH=false      # true=需要API Key才能访问
```

**修改后需要重启服务**（托盘右键 → 重启服务）

## 14. 常见问题

### ❓ 点击图标没有任何反应

**原因：** Python 未安装或依赖未装完

**解决：**
1. 右键 `openclaw_debug.bat` → 以管理员身份运行
2. 查看黑窗中的错误信息
3. 或重新运行 `install_full.bat`

---

### ❓ 浏览器能打开，但语音无法使用

**原因 1：** 浏览器未授权麦克风

**解决：**
- 地址栏左侧点击锁形图标
- 将麦克风设置为"允许"
- 刷新页面

**原因 2：** 使用的是 HTTP（8766端口），Chrome/Safari 不允许 HTTP 调麦克风

**解决：** 改用 HTTPS 地址：`https://本机IP:8765/app`（首次访问点击"继续访问"）

---

### ❓ 语音识别不准确

**解决方案：**
1. 设置 → 语音 → 将 STT 模型改为 `small` 或 `medium`
2. 减少背景噪音
3. 靠近麦克风说话
4. 确保 STT 语言设置为 `zh`（中文）

---

### ❓ AI 回复很慢

**原因：** 当前使用的 AI 平台响应慢

**解决：**
1. 在设置中启用"速度优先"模式
2. 添加更多 AI 平台 Key（自动选最快的）
3. 推荐 Groq 平台（速度极快）

---

### ❓ 提示"未配置 AI 平台"

**解决：**
1. 打开设置 → AI 平台
2. 选择一个平台填写 API Key
3. 点击"测试连接"确认可用
4. 保存

---

### ❓ 端口被占用，无法启动

**解决（Windows）：**

打开命令提示符（管理员）：
```cmd
netstat -ano | findstr :8766
taskkill /PID 找到的进程ID /F
```

然后重新启动 OpenClaw

---

### ❓ 安装包下载依赖失败

**可能原因：** 网络问题或代理设置

**解决：**
1. 检查网络连接
2. 关闭 VPN/代理后重试
3. 手动运行：
```cmd
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

## 15. 卸载方法

### 方法一：控制面板卸载（推荐）

1. Windows 设置 → 应用 → 已安装应用
2. 找到"OpenClaw AI 语音助手"
3. 点击"卸载"

### 方法二：手动删除

1. 删除安装目录（如 `C:\Program Files\OpenClaw`）
2. 删除桌面图标
3. 删除开始菜单项（`C:\Users\用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\OpenClaw AI 语音助手`）
4. 删除注册表开机自启（如果设置了）：
   - 按 `Win+R` 输入 `regedit`
   - 找到 `HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run`
   - 删除 `OpenClawAI` 项
