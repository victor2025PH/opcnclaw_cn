# 桌宠指挥官 — 产品交互重构方案

> 核心理念：桌宠不是装饰品，它是52个AI的指挥官
>
> 用户只需要和一只小龙虾说话，背后52个AI替你干活

---

## 一、为什么要重构

### 现在的问题

```
用户面对3个不相关的界面：
  1. 浏览器聊天窗口 → 打字聊天
  2. 桌宠小球 → 纯装饰，无互动
  3. Agent 团队 → 藏在 API 里，用户不知道

结果：用户觉得这只是另一个 ChatGPT，没有差异化
```

### 应该是什么样

```
用户桌面上有一只小龙虾 🦞
  ↓
双击它 → 弹出对话气泡
说一句话 → 小龙虾召唤团队
看到团队头像飞出 → 各自去工作
小龙虾表情变化 → 监工中/思考中/开心
工作完成 → 小龙虾递上结果卡片
  ↓
用户全程只跟一只小龙虾交互
```

## 二、桌宠的 6 种状态

```
😴 打盹    — 空闲，偶尔眨眼，呼吸动画
😊 待命    — 用户靠近/移动鼠标，抬头看
🎤 倾听    — 用户说话中，耳朵竖起，身体轻微膨胀
🤔 思考    — 分析任务，头上冒问号，转圈
👔 指挥    — 调度团队，身边飞出小头像
🎉 完成    — 结果出来，欢快跳动，递出文件

特殊状态：
😡 生气    — 服务器报错
😰 紧张    — 内存/CPU 过高
💬 聊天    — 简单对话直接回答
📢 提醒    — 微信新消息/定时任务到期
```

## 六、为什么这比聊天窗口更好

| 维度 | 传统聊天窗口 | 桌宠指挥官 |
|------|------------|----------|
| **感知** | 又一个 ChatGPT | 桌面上活的小龙虾 |
| **互动** | 打字→回复 | 说话/拖拽/右键/双击 |
| **情感** | 冷冰冰的文字 | 有表情有动作 |
| **存在感** | 需要打开浏览器 | 一直在桌面上 |
| **传播** | 截图没特色 | 截图/录屏自带传播力 |
| **团队感** | 看不到团队 | 小头像飞出飞入 |
| **差异化** | 和竞品一样 | **全球独创** |

## 七、实现路径

### 阶段 1：让现有桌宠活起来（2 天）

```
当前：/pet 页面是静态小球
目标：
  1. 集成到 Tauri 窗口（always on top，透明背景）
  2. 右键菜单（快速任务+设置+退出）
  3. 双击弹出输入框
  4. 状态同步（idle/listening/thinking/speaking）
```

### 阶段 2：团队组建面板（2 天）

```
点击快捷任务或输入需求 → 弹出团队面板
  显示 Agent 列表 + 任务编辑 + 开始按钮
  执行中显示实时进度
  完成后显示结果卡片
```

### 阶段 3：智能通知（1 天）

```
微信消息 → 桌宠弹通知
定时任务到期 → 桌宠提醒
系统异常 → 桌宠变红
每日早报 → 桌宠主动汇报
```

### 阶段 4：拖拽交互（1 天）

```
拖拽文件/文本/图片到桌宠
→ 自动识别类型
→ 弹出操作选项
→ 调用对应 Agent 处理
```

## 八、这才是真正的护城河

**ChatGPT 是一个输入框。**
**十三香是桌面上的一只活的小龙虾，身后站着 52 个 AI。**

没有任何竞品有这个体验。这不是功能差异，是**品类差异**。

用户会说："我桌面上有只小龙虾，我跟它说写个方案，它就召唤了 10 个 AI 帮我搞定了。"

这句话自带传播力。
